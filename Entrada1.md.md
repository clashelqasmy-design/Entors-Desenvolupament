# Entrada 1: com he creat i utilitzat GitHub

En aquesta entrada explico com he creat i utilitzat GitHub fins ara.

Primer de tot he creat un compte a GitHub posant el meu correu, un nom d’usuari i una contrasenya. Després he confirmat el correu i ja he pogut entrar.

A continuació he creat un repositori nou que es diu blog. L’he posat públic i he afegit un README.

També he activat GitHub Pages des de la configuració perquè el repositori es pugui veure com una web.

Per escriure les entrades he utilitzat fitxers .md amb Obsidian, que em serveix per editar millor aquest tipus de documents.

Ara ja sé crear repositoris, pujar fitxers i tenir una web publicada amb GitHub.
